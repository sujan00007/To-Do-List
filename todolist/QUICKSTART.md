# Quick Start Guide

## Run the Application

```bash
cd todolist
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Features Overview

- ✅ **Add Tasks**: Enter text and click Add
- ✅ **Edit Tasks**: Click edit icon on any task
- ✅ **Complete Tasks**: Click checkbox to mark done
- ✅ **Delete Tasks**: Click trash icon
- ✅ **Filter**: All / Active / Completed
- ✅ **Dark Mode**: Toggle in navbar
- ✅ **Persistent**: Data saved in localStorage
- ✅ **Responsive**: Works on mobile & desktop

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Lucide React Icons
- LocalStorage API
